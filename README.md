# Manipulação de Strings em Python

Repositório com exercícios desenvolvidos para praticar manipulação de textos e lógica de programação utilizando Python.

## Exercícios

### 1. Cadastro de Alimentos
Programa que recebe o nome de uma fruta, verdura e legume e exibe as informações na tela.

### 2. Cadastro de Produtos e Preços
Programa que recebe nomes e preços de produtos e exibe os valores formatados.

### 3. Contador de Vogais e Espaços
Programa que conta a quantidade de vogais e espaços presentes em uma frase.

### 4. Manipulação de Nome
Programa que exibe um nome em letras maiúsculas, minúsculas e invertido.

## Assuntos praticados

- Strings
- Métodos de texto
- Entrada e saída de dados
- Contagem de caracteres
- Fatiamento de strings
- Formatação de texto
- Lógica de programação
